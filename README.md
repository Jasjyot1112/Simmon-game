# 🎮 Simon Game

A fully functional Simon Game built using HTML, CSS, and JavaScript. The game tests your memory by generating a sequence of colors that the user must repeat correctly.

---

🚀 Features

* Dynamic sequence generation
* Memory-based gameplay
* Level progression
* High score tracking
* Interactive UI with animations

---

🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

🎮 How to Play

1. Press any key to start the game
2. Watch the sequence of flashing colors
3. Click the buttons in the same order
4. Sequence increases every level
5. Game ends if you click the wrong button
